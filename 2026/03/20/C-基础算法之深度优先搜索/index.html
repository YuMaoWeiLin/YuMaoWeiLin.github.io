深度优先搜索，英文叫 Depth First Search，通常简称 DFS。
它是一种非常基础但非常重要的搜索算法，在图论、迷宫、排列组合、回溯等问题中都有广泛应用。

DFS 的核心思想可以概括为一句话：

先沿着一条路一直走到底，走不通了再回头

一、什么是深度优先搜索

可以把 DFS 理解成在一个岔路很多的迷宫里找路

假设你从起点出发：

  1. 先选择一条路往前走
  2. 如果还能继续走，就继续往前
  3. 直到走不动为止
  4. 然后回退到上一个路口，换另一条路继续尝试

二、DFS 的基本特征

深度优先搜索通常有以下几个特点：

  • 一条路走到底
  • 走不通时回溯
  • 常用递归实现
  • 适合枚举所有可能情况

因此，DFS 经常和“回溯”一起出现。

三、DFS 的经典应用

DFS 常见应用包括：

  • 遍历图或树
  • 迷宫寻路
  • 枚举全排列
  • 枚举子集
  • 连通块统计
  • 八皇后问题
  • 数独求解

六、代码讲解

1. step 表示当前填到第几位

比如 step = 1，表示现在要填排列的第一位

2. used[i] 表示数字 i 是否已经被使用过

如果已经放进排列里了，就不能重复使用

3. 递归终止条件

说明前 n 个位置都已经填完，此时就得到了一个完整排列，可以输出

4. 回溯

这一步非常重要。
递归回来以后，要把刚才做过的选择撤销掉，方便尝试其他情况

七、DFS 与 BFS 的区别

很多初学者会把 DFS 和 BFS 弄混

DFS 是：

  • 一条路走到底
  • 常用递归
  • 更像“钻进去”

BFS 是：

  • 一层一层扩展
  • 常用队列
  • 更像“向四周扩散”
